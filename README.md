# ShakeSphere 🎭

**ShakeSphere** is my little homage project to William Shakespeare - exploring themes and motifs across plays through interactive graphs, and citing his influences on our language and modern-day pop culture. It also includes a "Kids Mode" to make it accessible and fun for younger readers!

## 🌟 Features

### 🕸️ The Motif Graph
A dynamic, interactive D3.js network graph that maps the relationships between Shakespeare's plays, their core themes (motifs), and modern-day adaptations.
- **Plays**: Explore tragedies, comedies, and histories.
- **Pop Culture**: See how *The Lion King* relates to *Hamlet*, or how Taylor Swift's *Love Story* connects to *Romeo & Juliet*.
- **Indian Cinema**: Discover Vishal Bhardwaj's acclaimed trilogy (*Haider*, *Omkara*, *Maqbool*).
- **Characters & Settings**: Trace the origins of iconic characters and their geospatial settings.

### 🧒 Kids Mode
A dedicated, interactive space designed to make the Bard accessible to young audiences.
- **AI-Powered Summaries**: Uses Gemini AI to explain complex plots in simple, fun language for kids.
- **Interactive Play Cards**: Fun emojis and simplified categories (Magic, Monsters, Big Adventures).

### 📖 Vocabulary & Archive
- **Word-Wizard**: Discover words invented by Shakespeare (like *swagger* and *eyeball*).
- **Live Archive**: Real-time scholarly insights and resources.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/iswaryam/shakesphere.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your environment variables:
   Create a `.env` file in the root and add your Gemini API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
